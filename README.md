#  سیستم سفارش آنلاین رستوران


- **نام و نام خانوادگی:** الهه سلطانی
- **شماره دانشجویی:** 03221058705046

سناریو:

فرض کنید یک رستوران آنلاین دارید.

وقتی مشتری سفارش ثبت می کند، آماده شدن غذا چند ثانیه زمان می برد.

بعد از آماده شدن غذا باید یک عملیات دیگر انجام شود (مثلاً اطلاع به مشتری).

نام غذا

این تابع باید:

پیام «سفارش در حال آماده سازی است» را نمایش دهد.

با استفاده از setTimeout به مدت 3 ثانیه آماده شدن غذا را شبیه سازی کند.  (شبیه سازی انتظار زمان آماده شدن غذا)

بعد از آماده شدن غذا، تابع Callback  را اجرا کند (مثلا این تابع پیام "غذا آماده شد" را چاپ می کند).
